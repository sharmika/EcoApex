This doc explains how to build the Apex project using the
App Java SDK, the Tomcat6 and Java 6.


Run Apex war file with Tomcat
=============================
1.copy Apex war file and place inside C:\Program Files\Apache Software Foundation\Tomcat 6.0\webapps folder.

2.Start tomcat server.

2.open url in any browser

http://localhost:8080/Ecotronyx/
